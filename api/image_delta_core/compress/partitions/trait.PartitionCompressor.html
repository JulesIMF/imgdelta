Expand description

Handles compression for a single partition, regardless of type.

Implementations exist for:

  • FsPartitionCompressor — mounts and runs the 8-stage pipeline, writing patch files into ctx.patches_dir.
  • BiosBootCompressor — reads raw bytes and uploads as a single blob.
  • RawPartitionCompressor — reads raw bytes and uploads as a single blob.
  • MbrCompressor — reads 440 MBR bytes and uploads as a single blob.

Patch bytes are never returned in memory. FS compressors write individual patch files to ctx.patches_dir; the orchestrator packs them into one archive after all partitions are done.
