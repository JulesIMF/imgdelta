Expand description

A single stage in the compress pipeline.

Each stage is a stateless struct that transforms FsDraft given the immutable shared StageContext. Stages are designed to be independently testable: a stage should depend only on the data it receives, not on any external global state.

Execute the stage.

Receives ownership of draft, transforms it, and returns it. The pipeline runner calls stages in order, chaining the output of each as the input to the next.
