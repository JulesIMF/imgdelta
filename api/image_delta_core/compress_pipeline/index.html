Module compress_pipeline 

Source
Expand description

Eight-stage stateless compress pipeline for one Fs partition.

Each stage is a standalone function that takes a FsDraft (plus optional helpers) and returns a transformed FsDraft. Stages are independently testable — no stage depends on the output of a stage it does not directly receive.
